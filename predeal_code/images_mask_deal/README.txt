本部分为关于通过mask去噪和共聚焦合成核心的预处理代码

由于数据集部分有mask部分没有，这部分需要处理，还有就是各类噪（字母噪，器具噪等）processImages.ijm是imagej软件使用的脚本代码

字母噪和乳腺区域选取主要步骤为（数据集都当作无自带mask处理）：
1.先进行找最大轮廓进行一边粗处理，然后将原图片通过imagej批处理，可以得到一阶段初始的mask，尺寸与原图相同，在经过一次保留最大白色连通区域，可得到最终版本的mask
这里注意，保留最大白色连通区域代码中包含一个将图像四周像素设置为黑色0的代码，这是由于很多数据集在四周有白色长条噪该部分噪有时候会和字母等其他噪在边缘连通
2.根据mask选取乳腺区域
这里的任何处理都没有改变mask和原图尺寸，在根据mask选取乳腺区域代码中为了保险仍采取了mask对比原image进行相同大小的放缩


local数据的字母噪有独特性
在处理完后便进行左右乳腺的区分，该部分采用中线划分后对比两边的黑色像素比例大小方式，local的字母噪均为255像素
接着划分mlo和cc，这里从对称度入手，计算了单侧病人的所有对称的取了平均值来区分，找分辨率过大图像也是相同处理方案

最后来根据一个阈值来自适应找乳腺区域（该方案需要处理干净字母等噪音），这里由于local数据集过于庞大便采用该方案，实际上也可以尝试通过找local数据集的mask然后最大保留白色区域即可


切片拼接处理，zstack系列为多层采样为单层的数据
切片拼接处理为进行切patch，选取其中最大值，均值，最小值三者方差最大的来拼接单层图片
zstack为进行共聚焦操作，将多层放缩到相同尺寸图片后，对各个像素点进行选最大的像素点和平均像素值点来合成单层图片
